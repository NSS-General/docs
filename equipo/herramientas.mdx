## Herramientas principales

| Herramienta | Uso | Acceso |
|-------------|-----|--------|
| **Slack** | Comunicación interna, interacción con Ark | nssabogados.slack.com |
| **Google Workspace** | Email, Calendar, Drive, Docs | Gmail corporativo |
| **Notion** | Wiki interna, bases de datos, documentación | notion.so |
| **Asana** | Gestión de proyectos y tareas | asana.com |
| **Piriod** | Facturación y suscripciones | piriod.com |
| **NSS App** | Aplicación interna del estudio | App propia |
| **SII** | Trámites tributarios | sii.cl |

## Slack — Canal principal

Slack es la herramienta central de comunicación. Ark opera desde Slack y responde a solicitudes del equipo.

**Regla clave**: Todas las conversaciones van en **hilos** (threads), nunca mensajes sueltos al canal.

## Google Drive — Estructura

Los documentos de clientes se organizan por portafolio:

```
📁 NSS Drive
  📁 Internacionales
    📁 [Cliente A]
    📁 [Cliente B]
  📁 Crypto
  📁 27 Bis
  📁 RRHH
  📁 Contabilidad
  📁 Financiero
  📁 Outsourcing Laboral
```

## Piriod — Facturación

Dos entidades configuradas:
- **NSS Limitada** — Entidad principal
- **NSS SpA** — Entidad secundaria

Las suscripciones se crean durante el onboarding según el MSA del cliente.
