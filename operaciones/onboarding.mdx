## Resumen

El onboarding es el proceso de ingreso de un nuevo cliente a NSS. Cubre desde la recepción del caso hasta que el cliente está completamente operativo en todos los sistemas del estudio.

## Pasos del proceso

### 1. Intake (recepción)

El caso llega por **email** (Gmail) o **Slack** (mensaje de un socio/abogado).

Ark extrae los datos clave:
- Nombre del cliente (persona o empresa)
- RUT
- Contacto (email, teléfono)
- Tipo de servicio / portafolio
- Socio responsable

### 2. Carpeta en Google Drive

Se crea la estructura de carpetas del cliente dentro del portafolio correspondiente:

```
📁 [Nombre Cliente]
  📁 Contratos
  📁 Documentos
  📁 Correspondencia
  📁 Facturación
```

### 3. Proyecto en Asana

Se crea un proyecto de onboarding con las tareas específicas del portafolio:
- Checklist de documentos requeridos
- Hitos de seguimiento
- Responsables asignados

### 4. Usuario en NSS App

Se registra al cliente en la aplicación interna del estudio para seguimiento.

### 5. Facturación en Piriod

Se crea la suscripción del cliente en Piriod según el plan acordado (MSA).

### 6. Documentos requeridos

Cada portafolio tiene su lista de documentos. Ark hace seguimiento hasta que estén completos.

Desde Slack:

```

```

Ark te guiará paso a paso por el proceso completo.
